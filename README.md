# TASD-Enhanced Q216D Tokamak Simulation

Interactive 3D browser demonstration of ultra-precise tokamak energy transition control using the TASD framework.

## Features
- Real-time 3D toroidal plasma visualization
- TASD-based energy transition control (2.10108–2.10288 keV)
- Pellet injection and MGI event simulation
- Live energy and PCA charts

## How to run
Just open `index.html` in any browser.
