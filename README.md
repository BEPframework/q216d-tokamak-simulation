# Q216D Tokamak Simulation – TASD Enhanced v1.0

**Interactive 3D plasma visualization with ultra-precise energy transition control**

A fully interactive, browser-based 3D tokamak simulator powered by TASD (Toroidal-Attractor Signal Dynamics) attractor control. No installation required — runs entirely in the browser.

---

### Features

- Real-time Three.js torus visualization with 10 representation modes
- TASD-powered ultra-precise energy transitions (2.10108–2.10288 keV)
- Live energy, PCA, heating, and losses charts
- Real-time controls (temperature, density, zoom, speed)
- Pellet injection & MGI event simulation
- Pyodide backend + sound feedback

### How to Use

1. Open `index.html` in any modern browser (no installation needed).
2. Click **Start Simulation**.
3. Use the controls on the right to adjust plasma parameters.
4. Switch between visualization modes to explore different representations.
5. Trigger pellet or MGI events to observe transient dynamics.
