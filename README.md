# Q216D Tokamak Simulation – TASD Enhanced v1.0

Fully interactive 3D web-based tokamak simulator with TASD attractor control.

## Features
- Real-time Three.js torus visualization with 10 representation modes
- TASD-powered ultra-precise energy transitions (2.10108–2.10288 keV)
- Live energy/PCA/heating/losses charts
- Real-time controls (temperature, density, zoom, speed)
- Pellet & MGI event simulation
- Pyodide backend + sound feedback

## Quick Start
1. Open `q216d-tokamak-simulation.html` in any modern browser.
2. Click “Start Simulation”.
3. Use the controls on the right.
